Widgets Solaires Embarquables

Intégrez gratuitement des widgets Solar Data Atlas sur votre site web. Données actualisées, responsive et faciles à implémenter.

Comment utiliser ?

  1. 1. Sélectionnez le widget qui vous intéresse
  2. 2. Cliquez sur "Copier le code d'intégration"
  3. 3. Collez le code dans votre page HTML
  4. 4. Le widget s'affichera automatiquement

Nos Widgets Disponibles

Mini LCOE

Affiche le coût actualisé de l'électricité solaire (LCOE) en temps réel, comparé aux sources concurrentes.

Questions Fréquentes

Les widgets sont-ils gratuits ? ▼

Oui, tous nos widgets sont 100% gratuits et sans limite d'utilisation. Aucune clé API n'est requise.

Puis-je personnaliser le style du widget ? ▼

Les widgets sont optimisés pour s'adapter à votre design via leurs dimensions. Pour des personnalisations avancées, contactez-nous.

Les données des widgets sont-elles actualisées en temps réel ? ▼

Oui, les données (LCOE, tarifs, productivité) sont mises à jour quotidiennement ou à chaque chargement du widget.

Puis-je utiliser plusieurs widgets sur la même page ? ▼

Bien sûr ! Vous pouvez combiner autant de widgets que vous le souhaitez sur une même page.

Quel support technique est disponible ? ▼

Pour toute question ou problème, contactez-nous à bernard.pugnet13400@gmail.com. Nous vous aiderons rapidement.
